# Использование

## Установите зависимости

```
npm install
# or
yarn
```

## Для сборки скрипта и его запуска введите

```
npm run build
# or
yarn build
```

# Для разработки или оценки результата [живой странице](http://localhost:3001/) введите

```
npm start
# or
yarn start
```

# Для запуска [сервера](http://localhost:3000/) введите

```
npm run dev
# or
yarn dev
```
